## Commonly Used Git Commands
- git status
	- Shows the current status of the branch (including staged, unstaged, and other files)
- git add [file|*]
	- Stages a file
- git commit -m ""
	- Makes a commit with a message with the staged files
- git push
	- Pushes staged commits to origin
- git branch [branch] / git checkout -b [branch]
	- git branch [branch] does NOT switch to the new branch automatically
	- git checkout -b [branch] does switch to the new branch automatically
- git switch [branch]
	- Changes the currently selected branch
- git remote add origin [branch]
	- Pushes a local branch to origin/remote
